# 代表的なパソコンを構成するパーツ

パソコンを**作業場とその中にいる人間**に例えて比較すると、

| パソコン | 作業場の人間 |
|:---:|:-------:|
| CPU | 人間の脳 |
| メモリ | 作業台 |
| HDD / SSD | 倉庫 |

この3つでノートパソコンの性能が決まります。
